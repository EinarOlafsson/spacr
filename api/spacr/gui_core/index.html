spacr.gui_core.check_src_folders_files(settings, settings_type, q)

Checks if ‘src’ is a key in the settings dictionary and if it exists as a valid path. If ‘src’ is a list, iterates through the list and checks each path. If any path is missing, prompts the user to edit or remove invalid paths.

spacr.gui_core.initiate_root(parent, settings_type='mask')

Initializes the root window and sets up the GUI components based on the specified settings type.

Args:

parent (tkinter.Tk or tkinter.Toplevel): The parent window for the GUI. settings_type (str, optional): The type of settings to be displayed in the GUI. Defaults to ‘mask’.

Returns:

tuple: A tuple containing the parent frame and the dictionary of variables used in the GUI.
