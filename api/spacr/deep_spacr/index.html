spacr.deep_spacr.evaluate_model_performance(model, loader, epoch, loss_type)¶

Evaluates the performance of a model on a given data loader.

Args:

model (torch.nn.Module): The model to evaluate. loader (torch.utils.data.DataLoader): The data loader to evaluate the model on. loader_name (str): The name of the data loader. epoch (int): The current epoch number. loss_type (str): The type of loss function to use.

Returns:

data_df (pandas.DataFrame): The classification metrics data as a DataFrame. prediction_pos_probs (list): The positive class probabilities for each prediction. all_labels (list): The true labels for each prediction.
